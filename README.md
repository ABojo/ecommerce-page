# Eccommerce Page

This is an ecommerce product page built with React, TypeScript, and SCSS.

## Features

- Responsive design
- Cart implemented
- Quantity and add to cart controls implemented
- Mobile and desktop navigation
- Users can enlarge product images on larger screens
